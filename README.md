🎓 Student Data Manager

A Python-based Student Record Management System developed using Object-Oriented Programming (OOP) concepts. This project helps manage student information efficiently through a simple menu-driven interface.

🚀 Features

- ➕ Add New Student Records
- 🔍 Search Student by ID
- ✏️ Update Existing Records
- ❌ Delete Student Records
- 📋 Display All Student Records

🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- VS Code
- Git & GitHub

📚 OOP Concepts Implemented

- Classes & Objects
- Constructors ("__init__")
- Methods
- Encapsulation
- Menu-Driven Programming

📂 Project Structure

Student-Data-Manager/
├── student_manager.py
├── README.md

▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run:

python student_manager.py
